# SECOND BRAIN

Personal Knowledge & Thought Management System.

## Project Structure

```text
second-brain/
  backend/
  frontend/
  docs/
```

## MVP Scope

- Quick capture for notes
- Notes CRUD API
- Auto-tagging with rules
- Timeline view
- Search
- Task extraction

## Tech Stack

- Frontend: Next.js + Tailwind CSS
- Backend: FastAPI
- Database: MongoDB
